# STANDALONE AGENCY STANDARDS

## ARCHITECTURE & SPEED
- STACK: Vite + React + Tailwind + Supabase. NO Builders.
- MODULAR: No single file over 150 lines. Use "Islands" (Components).
- MEDIA: Use WebP/AVIF, Lazy-loading, and "Blur-up" placeholders.

## VISUAL FIDELITY (Harmony/Vibe Standards)
- LAYERING: Use overlapping elements and Z-index depth.
- MOTION: Use GSAP ScrollTrigger for "reveal" animations.
- TYPOGRAPHY: Use fluid clamp() scaling for high-end feel.

## INDUSTRY STYLE GUIDE
- AUTHORS: "Ivory Parchment" – Serif fonts, high white space.
- CANNABIS: "Botanical Minimalist" – Deep greens, sage, glass effects.
- PHOTOGRAPHY: "The Gallery" – Ultra-minimalist (B/W), edge-to-edge grids.
- COMEDY/ENT: "Neon Underground" – Dark mode, Electric Purple/Yellow.
- BARS/NIGHTLIFE: "Smoked Luxury" – Deep ambers, leather, gold.

## BACKEND & WIZARD
- DYNAMIC: Fetch all text/prices/images from Supabase.
- WIZARD: Build a /setup route to input API keys (Google, Anthropic, Resend).
- GOOGLE SYNC: Include a "Sync to Google Business" button in /admin.
